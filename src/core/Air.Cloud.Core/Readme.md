zh-cn:
Air.Cloud.Core 是一个基于 .NET 6 的类库，提供了 Air.Cloud 生态系统的核心功能。
它包含了各种 Air.Cloud 应用程序和服务共享的基本服务、实用工具和模型。
## 功能特性
- 基于SPI机制的分布式框架核心服务,通过定义标准,实现同类型中间件的可插拔替换,如分布式锁、配置中心等
- 支持分布式环境下的多种中间件,如消息队列、缓存等
- 动态应用挂载,允许使用Air.Cloud的服务挂载基于Air.Cloud.Core开发的其他任意服务
- 与其他 Air.Cloud 组件的集成,比如:天镜安全认证(正在开发中)
## 快速开始
要在项目中使用 Air.Cloud.Core，请参阅 [Air.Cloud 文档](http://airmeta.org.cn)。


en-us:
Air.Cloud.Core is a .NET 6 class library that provides core functionalities for the Air.Cloud ecosystem.
It includes essential services, utilities, and models that are shared across various Air.Cloud applications and services.


## Features
- Core services based on SPI mechanism for distributed frameworks, enabling pluggable replacements for similar middleware such as distributed locks and configuration centers.
- Support for various middleware in distributed environments, such as message queues and caching.
- Dynamic application mounting, allowing Air.Cloud services to mount other services developed based on Air.Cloud.Core.
- Integration with other Air.Cloud components, such as SkyMirror security authentication (under development).

## Getting Started
To use Air.Cloud.Core in your project, refer to the [Air.Cloud Documentation](http://airmeta.org.cn).
